**2016.12.05**

1. 本地创建并checkout branch1分支
git branch -b branch1
等效于
git branch branch1
git checkout branch1

1. 本地各种修改文件操作
git add .
git commit . -m="xxx"

1. 本地分支提交远程，提交后远程，可以在agile上编译发布这个分支
git push origin branch1

1. 本地回到主干
git checkout master

1. 本地合并分支
git merge branch1

1. 删除本地分支
git branch -d branch1

1. 进入远程gitweb，找到branch1分支，对比差异，远程合并master
远程和本地的merge单独各自进行

1. 删除远程分支（如有必要的话）
git branch -D branch1
